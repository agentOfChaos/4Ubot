4Ubot
=====

This bot applies sentiment analysis during an IRC chat session, and kicks users responsible of "unhappy" messages.

Running
-------

$ twistd -n tw4u

Dependancies
------------

[textblob](https://github.com/sloria/TextBlob>)
twisted

Which can be both installed via pip.
