4Ubot
=====

4U bot applies sentiment analysis during an IRC chat session, and kicks users responsible for "unhappy" messages.

Running
-------

$ twistd -n tw4u

Dependancies
------------

[textblob](https://github.com/sloria/TextBlob>)   
twisted   

(Both installable via pip)
